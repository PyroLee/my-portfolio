# 交互式简历网站 — 详细开发计划

## 设计思路概述

### 核心理念：用"讲故事"的方式展示简历

传统简历是一张纸，HR 平均只花 **6 秒** 扫一眼。我们的目标是让这 6 秒变成 **60 秒的沉浸式体验**。

借鉴 Apple 产品页的叙事手法：不是一次性把所有信息堆在屏幕上，而是**随着用户的滚动，一幕幕揭开你的故事**。每一次滚动都是一个新的"惊喜时刻"，让 HR 不自觉地继续往下看。

### 视觉策略 (Claude 风格提炼)

从 Anthropic/Claude 官网中提炼出的设计 DNA：

| 设计元素 | Claude 风格特征 | 我们的应用 |
| :--- | :--- | :--- |
| **配色** | 温暖的米白底色 `#FAF9F6`，深棕/暖黑文字 `#1A1A1A`，标志性的赭橙点缀 `#D97757` | 用作全局底色、正文字色、以及关键按钮/链接的高亮色 |
| **字体** | 大字号标题（60-80px）与精细正文的强烈对比 | Hero 区大标题冲击，正文用 16-18px 保证阅读舒适 |
| **留白** | 极致的大面积留白，内容不拥挤 | 每个模块之间有充足的 padding（100-150px） |
| **圆角** | 温和但不过分圆润的卡片圆角 | 使用 12-16px 圆角，传递亲和力 |
| **动效** | 克制、优雅、丝滑的过渡 | 不追求花哨，而是追求"恰到好处"的高级感 |

### 动效策略 (Apple 风格提炼)

从 Apple iPhone/MacBook 产品页中提炼出的交互核心：

1. **滚动驱动动画 (Scroll-Driven Animations)**：页面内容**不是页面加载后就全部出现**，而是 HR 向下滚动时，元素从透明/偏移位置丝滑地滑入可视区域。
2. **视差效果 (Parallax)**：不同层级的元素以不同速度移动，营造深度感和高级感。
3. **Sticky（粘性）叙事段落**：某些关键标题"钉"在屏幕上不动，背景内容在其下方滚过，像 PPT 翻页一样展示信息。
4. **非线性缓动 (Ease-Out Cubic)**：所有动画使用 `cubic-bezier(0.16, 1, 0.3, 1)` 等非线性曲线，模拟真实物理运动，让动效感觉"有重量"。

## 技术方案

### 技术栈确认

| 技术 | 选型 | 理由 |
| :--- | :--- | :--- |
| 框架 | **Next.js 15 (App Router)** | Vercel 原生支持，SSR/SSG 保证 SEO 和极致加载速度 |
| 样式 | **Vanilla CSS (CSS Modules)** | 最高自由度，精确控制每一帧动画 |
| 字体 | **Google Fonts (Inter)** | Claude 同款字体族，知性优雅 |
| 动画 | **CSS `@keyframes` + Intersection Observer API** | 零依赖实现 Apple 级滚动揭示动画，无需引入第三方库 |
| 部署 | **Vercel** | 与 Next.js 同源，零配置 CI/CD |
| 代码托管 | **Github** | 行业标准 |

> [!NOTE]
> **为什么不用 GSAP/Framer Motion 等动画库？** 
> 对于我们当前的两页简历站，纯 CSS 动画 + 浏览器原生 Intersection Observer 已经足够实现需求中描述的所有效果（滚动揭示、淡入、位移等），且零依赖意味着更快的加载速度和更小的包体积。如果后续你想加入更复杂的逐帧控制动画，我们可以随时引入 GSAP。

### 第二阶段：主页核心模块开发 🎨
**预计耗时**：30 分钟

按照 HR 阅读的滚动顺序，主页由以下模块组成（从上到下）：

#### 2.1 导航栏 (Navbar)
- 半透明毛玻璃效果 (backdrop-filter: blur)，固定在页面顶部
- 包含：姓名/Logo（左）、页面锚点链接（中）、联系我按钮（右）
- 滚动时会自动收窄并增加阴影

#### 2.2 Hero 首屏区 (Hero Section)  ⭐ 最重要
- 全屏高度 (100vh)，第一眼冲击力
- **大字体**：你的名字 + 一句话定位（如："数字化转型实践者"）
- **入场动画**：页面加载后，文字和装饰元素从淡出透明逐步出现，带有轻微的上浮位移，模拟 Apple 发布会的"文字升起"效果
- **底部暗示**：一个优雅的向下滚动引导箭头，带有持续的上下悬浮动画

#### 2.3 关于我 / 个人概述 (About)
- 滚动进入视口时触发 ScrollReveal 动画
- 左图右文（或上图下文）布局
- 简洁的自我介绍段落

#### 2.4 技能展示 (Skills)
- 使用标签云 (Tag Cloud) 或圆角轻量卡片展示核心能力
- 每张技能卡片滚入时，从底部渐入并带有交错延迟（第一张先出现，第二张延迟 0.1s 出现...营造"瀑布流"视觉）

#### 2.5 工作经历 / 项目长廊 (Experience)
- **垂直时间轴设计 (Vertical Timeline)**
- 每段经历是一个玻璃质感卡片 (Glassmorphism)
- 卡片交替出现在时间轴左右两侧
- 滚动时每张卡片独立触发 ScrollReveal（从左/右侧滑入）
- 鼠标悬停时，卡片会轻微上浮并出现柔和的边框高光

#### 2.6 页脚 (Footer)
- 极简风格，仅包含版权信息和社交链接

你在这一步需要做的：**我写完代码后，你在浏览器里查看效果，告诉我哪里需要调整（如文案、色彩等）。**
